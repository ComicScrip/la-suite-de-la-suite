Exercice

  1. Remarques que chaque nombre dans la suite est la somme des deux nombres précédents. Autrement dit, si on veut obtenir le n-ième nombre de la suite, il faut faire la somme des deux précédents : (n-1) et (n-2).
  2. Traduire ce pseudo code en js (fibo.js) : 
    Fonction Fibonacci(n: entier): entier;
        
Debut
    Délarer une variable "résultat" qui sera un entier;

    Si(n = 0) Alors
        résultat := 0;
    Sinon Si(n = 1) Alors
        résultat := 1;
    Sinon
        résultat := Fibonacci(n - 1) + Fibonacci(n - 2);
    FinSi
      
    Renvoyer résultat;
Fin
